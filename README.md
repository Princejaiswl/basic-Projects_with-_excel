This project focuses on analyzing store data using Excel. It involves extracting insights from sales, inventory, and customer data to improve more bussiness setup's and selling.

Features are :

1).Data cleaning and preprocessing.

2).Sales trend analysis.

3).Inventory management insights.

4).Customer behavior analysis.

5).Visualization of key metrics.

The final dashboard sheet provides a summarized view of key store performance metrics. It includes:

Sales Overview: Total revenue, monthly trends, and category-wise sales distribution.
Inventory Insights: Stock levels, best-selling products, and restocking alerts.
Customer Behavior: Purchase frequency, average spending, and loyalty analysis.
Visual Representations: Charts and graphs for quick insights of store.
